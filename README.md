# PHPMailer Registration Demo

PHPMailer Registration Demo is a PHP example project that combines user registration with asynchronous or threaded email-delivery experiments using PHPMailer.

## Highlights

- Basic registration form
- PHPMailer-based email notification flow
- Background or threaded email-delivery experiment
- Useful as a prototype for separating email work from request-response handling

## Tech Stack

- PHP
- PHPMailer
- Pheanstalk
- Spatie Async

## Getting Started

1. Install dependencies with Composer.
2. Configure environment variables for database and mail sender settings.
3. Serve the PHP files with your preferred local PHP server.

## Environment Variables

- `DB_HOST`
- `DB_USER`
- `DB_PASSWORD`
- `DB_NAME`
- `MAIL_FROM_ADDRESS`
- `MAIL_FROM_NAME`

## Notes

This repository is a prototype and learning example. It should be reviewed and hardened before any production use.
